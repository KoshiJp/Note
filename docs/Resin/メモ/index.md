# Resin / 資料

## resin-admin 利用開始方法

1. `http://localhost:8080/resin-admin/`にアクセス。
1. `Don't have a login yet?`に`Username` `Password` `Re-enter password`を入力する。
1. `Create Configuration File`ボタンを押す。
1. 画面に表示される設定方法に従って設定ファイルを修正する。

## 設定

resin.xmlとresin.propertiesにて設定する。
resin.propertiesにて設定した変数はresin.xmlにて利用される。

## デプロイ

webappsの直下にwarファイルを配置すると展開される。

他にもresin-adminを使用して配置する方法がある。[要確認]
